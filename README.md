> A simple querybuilder for database, supporting arcgis http rest interface


## Install

```sh
$ npm install --save querying
```


## Usage

```js
var querying = require('querying');

querying('Rainbow');
```

```sh
$ npm install --global querying
$ querying --help
```

```sh
# creates a browser.js
$ npm run browser
```
